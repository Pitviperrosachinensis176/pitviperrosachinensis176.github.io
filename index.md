## 🖼️ How to classify flowers

You can identify flowers in three simple steps.

1. Open the application by double-clicking the desktop icon.
2. Click the "Browse" button inside the main window.
3. Select an image file from your computer using the file explorer window.
4. Click the "Analyze" button.
5. The software will display the name of the flower category on the screen.

The software sorts flowers into five distinct groups. If you provide an image that does not contain a flower, the software reports an error message. It works best with high-resolution images.

**Does the software need internet access?**
No, the software performs all analysis locally on your machine. You do not need an active internet connection to identify flowers.

**What file formats does it support?**
The software accepts standard image formats like JPEG, JPG, and PNG.

**The window looks blank after I click Analyze. What happened?**
The software processes large images for a few seconds. Please wait for the progress bar to finish before clicking other buttons.

**Can I upload multiple photos at once?**
This version supports one photo at a time. Process one image, note the result, and click browse to start the next.

**Where does the software store my images?**
The software reads your files but does not copy or save them to an external server. Your personal data remains on your computer.

## 💡 Tips for better results

For the best results, use clear photos of single flowers. Avoid blurry images or photos with many different flower types in the same frame. Place the flower toward the center of the image and ensure the lighting is even. This enables the model to see the features of the petals and leaves clearly.

## 📋 Troubleshooting steps

If you encounter issues, try these steps in order.

- Restart the application.
- Check if your image file is corrupted by opening it in your default photo viewer. 
- Reinstall the application if you see missing file errors.
- Ensure your antivirus software does not block the application. 
- Try a different photo to see if the problem persists with specific files.

## 📝 About this project

This project uses modern tools to teach computers how to recognize visual features. It includes several parts:

- Image preprocessing: This cleans your photo so the computer sees only the relevant parts.
- CNN model training: This part uses a large collection of labeled photos to learn what each flower looks like.
- Evaluation: This keeps the software accurate by measuring how often it identifies flowers correctly.
- Prediction: This applies the learned knowledge to your specific photos.

The project exists to make artificial intelligence tools easier to use for everyone, not just programmers.
